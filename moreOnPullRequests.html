More on Pull Requests

19 October 2015

So last time I wrote about Git Pull requests, I more or less covered the basics and had not really made any practical requests. I had just made one basic one without any messages or meaningful pieces of code. Something the other apprentices and I are trying to do more of is reviewing each others code and so I tried making some pull requests today. I created multiple branches and somehow managed to delete my master branch when trying to use git with Intellij, but I finally managed to make a pull request with a little piece of code and functionality that could be read easily by someone.

I am still making some formatting and whitespace errors that I am still working on, but I think I am getting the hang of it. I am going to try and use pull requests and make them as frequently as possible now, so I can maybe get some feedback as I go and not fall into bad habits.
