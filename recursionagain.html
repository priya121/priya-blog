More Recursion

08 December 2015

Recursion can be a tricky concept and I know I definitely struggled with it when I first started learning how to code. But it is simply a method calling itself with changing parameters that causes it to recurse. This can often be accomplished by a while loop which actually creates fewer stack frames, but can be a less elegant way of tackling a simple problem.

So the main driving factor as far as I can see is the clarity of your code and the length of your program (including how much data it needs to analyze and compute).

This is a great webiste that has a lot of great mini examples and ways to practice recursion.
